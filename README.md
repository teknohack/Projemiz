# 🚀 ModuKeys

🥇 Fikirden Kodlara Hackathonu 1.lik projesi

ModuKeys, 24 saatlik hackathon sürecinde geliştirdiğimiz dinamik ve 3 boyutlu e-ticaret deneyimi sunan bir web projesidir.

Kullanıcı odaklı yapısıyla;
- Kişiselleştirilebilir ürün deneyimi  
- Etkileşimli 3D arayüzler  
- Oyunlaştırılmış indirim kupon sistemi  
- Kullanıcı kayıt & giriş altyapısı  
- Admin paneli ve favoriler sistemi  

gibi uçtan uca e-ticaret özellikleri içermektedir.

Bu repo, hackathon sürecinde geliştirdiğimiz ve birincilik kazanan projenin kaynak kodlarını içermektedir.
